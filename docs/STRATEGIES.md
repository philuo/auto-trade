# 交易策略说明

## 系统架构概览

本项目包含**两套独立的交易系统**：

### 系统对比

| 特性 | DCA-Grid 策略 | 技术分析系统 |
|------|--------------|--------------|
| **状态** | ✅ 生产运行 | 🔄 开发完成，未集成 |
| **交易模式** | 仅现货 | 现货 + 永续合约 |
| **策略类型** | DCA定投 + 网格交易 | 事件驱动技术信号 |
| **支持币种** | BNB/SOL/XRP/ADA/DOGE | BTC/ETH/BNB/SOL/XRP/ADA/DOGE |
| **杠杆支持** | 无 | BTC 1-5x, ETH 1-3x |
| **运行方式** | `bun run strategy` | 未集成到主系统 |
| **文件位置** | `src/strategies/spot-dca-grid/` | `src/trading/technical-coordinator.ts` |

### 当前生产系统

**DCA-Grid 策略** 是当前实际运行的交易系统：

```

│         │                                                    │
```

## DCA-Grid 策略详解

### 策略配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| **交易模式** | 现货 (SPOT) | 不使用合约杠杆 |
| **支持币种** | BNB, SOL, XRP, ADA, DOGE | 不支持BTC/ETH |
| **Maker手续费** | 0.08% | 限价单手续费 |
| **Taker手续费** | 0.10% | 市价单手续费 |
| **更新频率** | 5秒 | 市场数据轮询间隔 |

### DCA 定投机制

| 触发条件 | 动作 | 说明 |
|---------|------|------|
| 价格下跌 ≥ 3% | 逆向买入 | 跌得越多，买得越多 |
| 每24小时 | 常规定投 | 固定金额买入 |

**逆向DCA示例：**

| 价格区间 | 买入金额 | 累计持仓 | 平均成本 |
|---------|---------|---------|---------|
| $100 | 100 USDT | 1.0 | $100 |
| $97 (-3%) | 150 USDT | 2.55 | $98.04 |
| $94 (-3%) | 200 USDT | 4.68 | $96.58 |
| $91 (-3%) | 250 USDT | 7.43 | $95.30 |

### 网格交易机制

| 参数 | 值 | 说明 |
|------|-----|------|
| 网格数量 | 20个 | 将价格区间分成20层 |
| 间距方式 | 等比 (geometric) | 每格按比例增长 |
| 单格订单 | 50 USDT | 每个网格的订单大小 |

**网格运行示例：**

```
当前价格: $100
网格区间: $80 - $120 (±20%)

网格分布:
$120 ━━━━━━━━━━━━━━━━━━ 卖出 (上限)
$115 ━━━━━━━━━━━━━━━━━━ 卖出
$110 ━━━━━━━━━━━━━━━━━━ 卖出
$105 ━━━━━━━━━━━━━━━━━━ 卖出
$100 ━━━━━━━━━━━━━━━━━━ 当前价格
$95  ━━━━━━━━━━━━━━━━━━ 买入
$90  ━━━━━━━━━━━━━━━━━━ 买入
$85  ━━━━━━━━━━━━━━━━━━ 买入
$80  ━━━━━━━━━━━━━━━━━━ 买入 (下限)

价格在区间内波动时，自动低买高卖
```

### 模式切换逻辑

| 市场状态 | 使用模式 | 判断依据 |
|---------|---------|---------|
| 下跌趋势 | DCA为主 | 价格持续下跌，积累筹码 |
| 震荡市场 | 网格为主 | 价格在区间内波动 |
| 上涨趋势 | 减持获利 | 网格逐步卖出 |

## 资金配置建议

### 保守型配置（适合新手）

| 资金分配 | 金额 | 说明 |
|---------|------|------|
| 现货交易 | 70% | 用于DCA-Grid策略 |
| 应急储备 | 30% | 不参与交易，应对极端情况 |
| 合约杠杆 | 0% | 完全不使用 |

### 资金分配示例（10,000 USDT）

| 用途 | 金额 | 币种分配 | 单币种 |
|------|------|----------|--------|
| 现货交易 | 7,000 | BNB/SOL/XRP/ADA/DOGE | 各1,400 |
| 应急储备 | 3,000 | USDT | - |

### 风险控制

| 风险指标 | 阈值 | 操作 |
|---------|------|------|
| 单币种最大仓位 | 30% | 超过不再买入 |
| 应急储备比例 | 20-30% | 必须保留 |
| 单币种止损 | 15-20% | 触发后停止交易 |
| 总回撤控制 | 20% | 超过暂停新开仓 |

## 风险等级对比

| 策略 | 最大亏损 | 爆仓风险 | 适合人群 |
|------|---------|---------|----------|
| 仅现货（DCA-Grid） | 被套30-50% | 无 | 所有新手 |
| 现货+网格 | 被套20-30% | 无 | 中级用户 |
| 1x合约网格 | 可能爆仓 | 低 | 高级用户 |
| 2x+合约 | **极易爆仓** | **高** | ❌ 不推荐 |

### 极端风险案例

| 场景 | 初始 | 跌幅 | 现货结果 | 5x合约结果 |
|------|------|------|----------|-----------|
| 正常下跌 | $10,000 | -20% | 亏损$2,000 | 亏损$10,000 **爆仓** |
| 插针 | $10,000 | -25%(瞬时) | 被套$2,500 | **立即爆仓** |
| 熊市 | $10,000 | -50% | 被套$5,000 | **早已爆仓** |

**结论：** 现货最安全，合约极易爆仓

## 熔断机制

### 紧急条件

| 条件 | 阈值 | 触发动作 |
|------|------|----------|
| 单K线涨跌幅 | > ±10% | 暂停新开仓 |
| 1小时波动率 | > 50% | 暂停新开仓 |
| 总回撤 | > 15% | 警告，暂停加仓 |
| 总回撤 | > 20% | 危险，考虑平仓 |
| 总回撤 | > 30% | 紧急平仓 |

### 应对流程

```
┌──────────────────┐
│ 检测到极端行情    │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│ 1. 暂停新开仓    │ ← 立即执行
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│ 2. 取消挂单      │ ← 5秒内执行
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│ 3. 评估持仓      │ ← 人工确认
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│ 4. 必要时平仓    │ ← 根据情况执行
└──────────────────┘
```

## 技术分析系统说明

### 系统特点

| 特性 | 说明 |
|------|------|
| **信号类型** | 15种技术信号（MA交叉、RSI、MACD、布林带等） |
| **驱动方式** | 事件驱动（只在穿越时产生信号） |
| **验证方式** | 基于历史统计的置信度计算 |
| **支持模式** | 现货 + 永续合约 |

### 合约支持（仅技术分析系统）

| 币种 | 杠杆范围 | 说明 |
|------|---------|------|
| BTC | 1x - 5x | 仅整数倍 |
| ETH | 1x - 3x | 仅整数倍 |
| 其他 | 不支持 | 仅现货 |

**重要：** 技术分析系统虽然支持合约，但**尚未集成到主运行脚本**，不能直接使用。
