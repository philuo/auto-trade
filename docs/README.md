# OkX 量化交易系统 - 文档目录

## 项目范围

本项目支持以下两种交易模式：

- **现货交易（SPOT）**: 使用 `cash` 模式，无杠杆
- **永续合约（SWAP）**: 使用 `isolated` 逐仓模式，支持杠杆

### ⚠️ 业务规则限制

**重要约束**：
1. **禁止借币**: 不允许任何形式的借币操作
2. **杠杆限制**:
   - BTC 合约：最大 5x 杠杆
   - ETH 合约：最大 3x 杠杆
   - 其它币种：仅允许现货交易，不允许合约
3. **币种限制**: 仅允许交易以下 7 个币种（BTC、ETH、BNB、SOL、XRP、ADA、DOGE）

### 支持的交易币种

| 币种 | 现货 | 合约 | 最大杠杆 |
|------|:----:|:----:|:--------:|
| BTC | ✅ | ✅ | 5x |
| ETH | ✅ | ✅ | 3x |
| BNB | ✅ | ❌ | - |
| SOL | ✅ | ❌ | - |
| XRP | ✅ | ❌ | - |
| ADA | ✅ | ❌ | - |
| DOGE | ✅ | ❌ | - |

## ⚠️ 重要提醒

1. **严格遵守业务规则**:
   - **禁止借币**: 不允许任何形式的借币操作
   - **币种白名单**: 仅允许交易白名单内的币种
   - **杠杆限制**: BTC 合约 ≤ 5x，ETH 合约 ≤ 3x
   - **合约范围**: 仅 BTC 和 ETH 可做合约，其它币种仅现货
2. **交易范围**: 本项目仅支持现货（SPOT）和永续合约（SWAP），不支持币币杠杆、交割合约、期权等
3. **模拟盘优先**: 所有开发测试都在模拟盘进行，确认无误后再考虑实盘
4. **安全第一**: 不要将 API Key 提交到版本控制系统
5. **风险控制**:
   - 现货：无爆仓风险，但资金全额占用
   - 合约：有爆仓风险，需监控强平价格，严格控制杠杆倍数
6. **充分测试**: 在实盘使用前进行充分的测试
7. **监控告警**: 部署完善的监控和告警机制
